# Codex Codegen Notes

This directory stores repo-specific design decisions and coding rules that Codex must apply for future requests in this repository.

Read these files before making code or documentation changes:

- [repository-rules.md](repository-rules.md): module, dependency, configuration, documentation, and verification rules.
- [architecture-decisions.md](architecture-decisions.md): decisions already applied in the repository.
- [codex-workflow.md](codex-workflow.md): workflow rules for implementing future Codex changes.

These notes are part of the project architecture documents under `doc.arch`.
